## Como iniciar a aplicação

### server

#### rodar em desenvolvimento (porta 5000)
```npm install```

```npm run dev```

#### rodar em produção (porta 8080)
```npm install```

```npm start```

### web-app

#### rodar em desenvolvimento (porta 3000)
```npm install```

```npm start```

#### rodar em produção (porta 3000)
```npm install```

```npm run build```

```serve -s build```
